Rig Framework

Learn more

The Foundation of Agentic AI

Rig is an open-source Rust-based framework designed for building powerful, lightweight AI agents. It provides a high-performance foundation that abstracts away the complexity of juggling multiple models, databases, and external tools, turning them into a coherent development experience.

With Rig, developers can focus on innovating at the application layer rather than wrestling with custom integrations. The framework's robust architecture supports multi-agent orchestration, on-chain interactions, and enterprise-grade reliability.

Supported Model Providers

Rig offers a unified API across multiple AI model providers, allowing developers to choose the right model for each task or seamlessly switch between providers with minimal refactoring.

Core Capabilities

  • Type-safe AI agent development
  • Multi-agent orchestration
  • Streamlined prompt handling

Database Integrations

  • Vector databases (MongoDB, PostgreSQL, Qdrant)
  • Managed embedding storage
  • Retrieval-augmented generation

Advanced Features

  • Streaming API support
  • Blockchain/on-chain integrations
  • Model Context Protocol (MCP) support

From Vision to Reality

Arc is a thriving community of developers and non-developers coordinated towards building and distributing advanced practical AI applications and AI agents built with Rig. Through the Ryzome platform—the "Agentic App Store"—Arc connects intelligent agents with thousands of digital services, creating a unified economy for the next generation of AI.

By bridging AI with blockchain technology, Arc is transforming theoretical AI capabilities into practical automation that delivers tangible benefits to users across Web2 and Web3.
